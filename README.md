# ZX81-Screen-Editor
This tool allows you to create and edit artwork for display on the ZX81 computer.

<img width="2561" height="1462" alt="screely-1763323253076" src="https://github.com/user-attachments/assets/d679d8d1-0498-4c95-b8fb-b385287611f7" />

## Features
* Select characters from the character picker on the right and then click and drag to draw on the screen.
* Use the "Painting Mode" to draw using the block character.
* Your creations are automatically saved in your browser's local storage, or you can save and load then to files using the buttons below the editor window.
* Export your creations as PNG images or ASM code for use in your projects.

## Run
`npm run dev`
